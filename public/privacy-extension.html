2. Purpose of the Extension

The Extension enables authenticated Leadesk users to import publicly visible LinkedIn profile data as leads into their own Leadesk CRM via a single click, and to invoke AI-assisted features (Brand Voice analysis, message generation) directly from within the browser.

3. Data Processed

3.1 From the authenticated user

  • Leadesk session authentication token (JWT), read from the active tab on app.leadesk.de or staging.leadesk.de
  • Stored locally in chrome.storage.local to avoid repeated sign-in
  • User interaction events within the Extension (for feature execution, no tracking)

3.2 From imported LinkedIn profiles (lead data)

  • Name, job title, headline
  • Company, position, location
  • Public LinkedIn profile URL
  • About text, work experience, education, skills, languages (insofar as publicly visible on the profile)
  • Profile picture URL

Only data already visible to the logged-in user on the LinkedIn profile is captured. No background activity takes place — data is read solely upon explicit user action ("Import Lead").

4. Legal Basis (GDPR)

  • Article 6(1)(b) GDPR (contract performance) for processing of authentication tokens and transmission of imported leads to the user's Leadesk backend
  • Article 6(1)(f) GDPR (legitimate interest) for processing LinkedIn profile data for the purpose of B2B business development by the user

Lawful processing of imported lead data by the user (e.g. subsequent outreach) lies within the user's own responsibility as data controller vis-à-vis the data subject.

5. Recipients and Data Transfer

Data is transmitted exclusively to the user's Leadesk backend:

  • https://supabase.leadesk.de (production)
  • https://supabase-staging.leadesk.de (staging)

All servers are located exclusively in Germany (Hetzner Online GmbH, Falkenstein/Nuremberg data centers). No transfer to third countries occurs. No sharing with third parties occurs. Data is not sold, rented, or used for advertising purposes.

6. Retention

  • Authentication tokens: until user sign-out or token expiry (typically 60 minutes, with automatic refresh)
  • Imported lead data: stored within the user's Leadesk CRM per their settings. Deletable by the user at any time.

7. Data Security

All data transmission is encrypted via HTTPS (TLS 1.2+). The Extension uses no external tracking scripts, no advertising networks, and no third-party analytics services.
